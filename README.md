
#  Banking Business Intelligence Analysis

An end-to-end **Business Intelligence and Data Analytics** project in the banking domain that transforms raw banking data into actionable business insights using **Python**, **PostgreSQL**, **SQL**, and **Power BI**.

This project covers the complete analytics workflow—from data preprocessing and exploratory data analysis (EDA) to database management, SQL-based business analysis,interactive dashboard creation, and business driven visualizations

---

##  Project Objectives

- Clean and preprocess banking datasets using Python.
- Perform exploratory data analysis (EDA).
- Store cleaned data in PostgreSQL.
- Execute SQL queries to answer business questions.
- Build an interactive Power BI dashboard.
- Generate business insights to support decision-making.

##  Project Workflow

1. Collected banking datasets.
2. Cleaned and preprocessed the data using Python.
3. Performed Exploratory Data Analysis (EDA).
4. Loaded the cleaned dataset into PostgreSQL.
5. Wrote SQL queries to answer business questions.
6. Built an interactive Power BI dashboard.
7. Derived business insights from the visualizations.

---

##  SQL Analysis

Some of the business questions answered include:

- Display customer names and ages.
- Count total customers.
- Find customers with multiple credit cards.
- Analyze customer occupations.
- Identify top customers by bank deposits.
- Compare average income by occupation.
- Analyze customer nationality distribution.
- Find customers with high-value loans.
- Calculate average credit card balance.
- Identify customers with above-average estimated income.
- Calculate total customer assets.
